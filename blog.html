Question-1: What are the different ways to select element in the DOM?



Answer: There are several ways to select elements in the DOM (Document Object Model) using JavaScript. Here are the most common methods: using getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll in the DOM.


Question-2: What is the difference between innerHTML, innerText, and textContent ?



Answer: The innerHTML, innerText, and textContent properties are all used to manipulate the content of an HTML element.


Question-3: What is event delegation in the DOM?



Answer: Event delegation allows us to handle events on a parent element instead of attaching them to multiple child elements. it helps us to make dynamic elements


Question-4: What is event bubbling in the DOM?



Answer: Event bubbling is a concept in the Document Object Model (DOM) where an event, such as a click or a keypress, propagates (or "bubbles") up from the target element to the root of the document.
